# 🏥 SafeOp — Sistema de Seguridad Quirúrgica

## 📌 Descripción

SafeOp es una plataforma web diseñada para mejorar la seguridad en procedimientos quirúrgicos mediante la digitalización del checklist quirúrgico, el control de instrumentación y la trazabilidad completa de cada cirugía.

El sistema permite gestionar múltiples clínicas, registrar cirugías, ejecutar el checklist en tiempo real y validar el conteo de instrumentos, reduciendo riesgos dentro del quirófano.

---

## ⚠️ Problemática

En muchos entornos clínicos, los procesos quirúrgicos aún dependen de registros manuales o poco estructurados, lo que puede generar:

- Errores en la identificación del paciente  
- Fallos en la verificación del procedimiento  
- Pérdida o mala contabilización de instrumentos  
- Falta de trazabilidad y seguimiento  

SafeOp busca reducir estos riesgos mediante una solución digital centralizada, simple y accesible.

---

## 🚀 Funcionalidades principales

- Gestión de clínicas y cirugías  
- Ejecución del checklist quirúrgico (Sign In, Time Out, Sign Out)  
- Registro y validación de conteo de instrumentos  
- Generación de alertas en caso de inconsistencias  
- Historial completo de cirugías  
- Sistema multiusuario con diferentes roles  

---

## 👥 Roles del sistema

- **Supervisor**: acceso general y supervisión  
- **Coordinador**: gestión de cirugías  
- **Encargado**: ejecución del checklist  
- **Consulta**: visualización de información
